Welcome

The physics core now runs inside a dedicated Web Worker. Rapier’s WebAssembly module instantiates a two-part hopper from the shared genome schema, while the main thread renders both the live simulation and a morph preview grid with Three.js. Use the control button to pause or resume the simulation loop.

Serve this directory with python3 -m http.server 8080 and open http://localhost:8080/index.html to explore the prototype.
